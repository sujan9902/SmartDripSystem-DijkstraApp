## 📖 Overview

SmartDripSystem-DijkstraApp is a smart irrigation management platform that uses **Dijkstra's Shortest Path Algorithm** to determine the most efficient routes for water distribution across agricultural fields.

The application visualizes irrigation networks as weighted graphs, allowing users to analyze, optimize, and manage water delivery efficiently.

* 💧 Efficient water distribution planning
* 📊 Interactive graph visualization

* 🔄 Dynamic network updates

## 🧠 Algorithm

The system uses **Dijkstra's Algorithm** to calculate the shortest path between irrigation nodes.

### Benefits

* Finds optimal water delivery routes
* Reduces water wastage
* Improves irrigation efficiency
* Supports scalable irrigation networks

## 🚀 Getting Started

### Prerequisites

* Node.js (v18+ recommended)
* npm or yarn

### Clone the Repository

```bash
git clone https://github.com/sujan9902/SmartDripSystem-DijkstraApp.git
cd SmartDripSystem-DijkstraApp
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Application will be available at:

```bash
http://localhost:5173
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
